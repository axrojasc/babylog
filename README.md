BabyLog es una aplicación móvil híbrida desarrollada con Ionic + Angular + Capacitor, diseñada para gestionar información y resgistros asociados al cuidado y seguimiento de bebés.

Herramientas instaladas:
Node.js 20+
Ionic 7
Angular 17 (NgModules)
Capacitor
TypeScript
Firebase Firestore

Cómo ejecutar el proyecto

1. Clonar el repositorio:

git clone https://github.com/axrojasc/babylog.git
cd babylog

2. Instalar dependencias:

npm install

3. Ejecutar la aplicación en el navegador

ionic serve

Esto abrirá la app en:
http://localhost:8100
